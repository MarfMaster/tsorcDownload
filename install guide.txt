ESTIMATED TIME: 3 MINUTES

0.Download tModLoader from Steam: https://store.steampowered.com/app/1281930/tModLoader/

1.Open tModLoader and select Mod Browser, then search for The Story of Red Cloud or tsorc. Download and load the TSORC Revamp mod. 

1a.Optional but recommended: Download and load the Recipe Browser Mod.

2.On the main menu screen, click the text prompt to download the TSORC Music Mod.

3.Create a new character named Red Cloud.

4.On the world selection screen, select New, then click "Load Custom Map." Hit back. Then play!
